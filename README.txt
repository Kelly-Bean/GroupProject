###########README
# The purpose of this project is to analyze data
# for home sales in the Staten Island borough of New York City.
# 
# The file 'LS6_DataGathering.R' loads all of the available data
# into R for analysis, as a single object 'bk' along with taking
# an initial look at the data.
#
# The file 'LS6_DataCleaning.R' uses a variety of methods to inspect
# and then clean the data. 
#
# The file 'LS6_DataPresentation.Rmd' presents the implications of our analysis
#
